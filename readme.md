# Quicky Translate

*A fast, lightweight translation extension for Firefox (with unofficial Chromium support)*

Quicky Translate is a modern translation extension that brings instant translations to your browser. Clean interface, fast performance, and zero clutter—just select text and translate.

---

## Features

- **Clean Interface:** Minimal design that stays out of your way
- **Fast Translation:** Instant results as you browse
- **Multiple Engines:** Switch between Google Translate and Yandex
- **Flexible Options:**
  - Choose your target language
  - Toggle automatic translation
  - Customize translation behavior

***

## Privacy

- Only selected text is sent to translation services
- No personal data is collected or stored
- Translation engines receive only the text you want translated

Note: The extension cannot run on restricted pages like `addons.mozilla.org` and `support.mozilla.org` due to browser security policies.

***

## Installation

1. Download the latest release from the releases section
2. Install through your browser's extension manager
3. Start translating

## FAQ

**Why does it need access to all websites?**
To translate content on any webpage you visit.

**Does the extension track my browsing?**
No. Your data stays private—nothing is collected or stored.
